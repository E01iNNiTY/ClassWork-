Vocabulary Definitions

  • Submit Button: A button that sends data.
  • For Attribute HTML: Specifies when you bind data.
  • Checkboxes: A square box that can be checked off.
  • Fieldset: Groups different elements on a form.
  • Form Action Attribute: Specifies where to send data.
  • Legend: A caption for the fieldset element.
  • Post: Used to send data to a server.
  • Get: Used to send data to the same server.
  • Radio Button: A subset of buttons.
  • Text Area: An input element that allows multiple lines of text to be inputted.
