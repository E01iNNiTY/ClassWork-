Application Terminologies

Layers

a group of reusable components that are reusable in similar circumstances.

Authorization

The right or a permission that is granted to a system entity to access a system resource.

User Interface

The means by which the user and a computer system interact, in particular the use of input devices and software.

Application Software

An application program (software application, or application, or app for short) is a computer program designed to carry out a specific task.

Software

The programs and other operating information used by a computer.

Authentication

The process or action of verifying the identity of a user or process.

Error Handling

Error handling is the process of responding to and recovering from error conditions in your program.

Cross-Cutting Concerns

Cross-cutting concerns are parts of a program that rely on or must affect many other parts of the system. They form the basis for the development of aspects. Such concerns do not fit cleanly into object-oriented or procedural programming.

Persistence Layers

Any software layer that makes it easier for a program to persist its state.

Error Message

A message displayed on a monitor screen or printout indicating that an incorrect instruction has been given or that there is an error resulting from faulty software or hardware.

Logic Layer

the part of a program where classes determine the information needed to solve problems, manipulate that information, and return results to the presentation layer for

Logging

the act of keeping a log of events that occur in a computer system, such as problems, errors or just information on current operations
