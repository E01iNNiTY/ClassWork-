Week 9

CSS: display

The display property in CSS specifies how an element is displayed on the page. Common values include block, inline, flex, and grid.

Track Tag

The <track> HTML tag is used to define text tracks for <video> and <audio> elements, such as subtitles or captions.

CSS: visibility

The visibility property in CSS controls whether an element is visible or hidden. Values include visible, hidden, and collapse.

CSS: font-variant

The font-variant property in CSS specifies whether text should be displayed in small caps or normal text. Example values include normal and small-caps.

CSS Grid: grid-template-rows

The grid-template-rows property defines the row structure of a CSS Grid container. You can specify row sizes using units like px, %, or fr.

CSS Grid: fr Unit

The fr unit is a flexible length unit in CSS Grid. It represents a fraction of the available space within the grid container.

CSS Grid: grid-template-columns

The grid-template-columns property defines the column structure of a CSS Grid container. You can specify column sizes using units like px, %, or fr.

CSS: text-decoration

The text-decoration property in CSS specifies the appearance of text decorations, such as underline, overline, line-through, or none.

CSS Grid: grid-template-areas

The grid-template-areas property defines named grid areas within a CSS Grid container, allowing for easier layout placement using area names.

CSS Grid: grid-area

The grid-area property assigns an element to a specific named grid area defined by grid-template-areas.

noscript HTML Tag

The <noscript> HTML tag defines content that is displayed if the user's browser does not support JavaScript or has it disabled.
